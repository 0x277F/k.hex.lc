Some Pitfalls in Bruns and Herzog 3.3.26


1 November 2024

As far as I have seen, solutions available online for this exercise are either incorrect or incomplete. The problem (3.3.26 in Bruns and Herzog’s Cohen-Macaulay Rings) is:

Let (R,𝔪,k) be a Gorenstein local ring of dimension d, and M a finite module of finite projective dimension. Show that Tor ⁡ iR(k,M) = Ext ⁡ Rd−i(k,M) for all i.

A Solution if R is Regular

This approach works if we assume that R is a regular ring, but fails in any case where R is Gorenstein but not regular.

Proof. Let x = x1,…,xd be a regular sequence generating 𝔪, then consider the Koszul complex

K∙(x) : 0 →∧ dF →∧ d−1F →⋯ → F → R → 0

where F is a free module of rank d. By [BH98, Cor. 1.6.14], K∙(x) is a free resolution of k, which is acyclic with respect to the functor −⊗ M, so by definition Tor ⁡ i(k,M) = Hi(K∙(x) ⊗ M) = Hi(x,M), where Hi is the Koszul homology. However, since we can take an M-sequence y1,…,yi ∈𝔪 for i ≤ d, [BH98, Thm. 1.6.16] implies that Hd−i(x,M) = Ext ⁡ i(k,M), or, rephrased, Tor ⁡ i(k,M) = Ext ⁡ d−i(k,M). □

However, this technique is not adaptable to the case where R is Gorenstein but not regular. If we examine the Koszul complex K∙(x) where x is a maximal R-sequence, we obtain a free resolution of R∕(x), but not of k. Likewise, if we were to take x to be a generating set of 𝔪, it could not be a regular sequence. In particular, [BH98, Thm 1.6.17] would guarantee that the complex K∙(x,M) has nonzero homology at some order i > 0, so we would be incapable of forming a resolution of any sort.

My Solution

A more viable approach is to induct on projdim ⁡ M.

Proof. Let m = projdim ⁡ M – if m = 0, then M is projective, hence free (since R is local), and k ⊗ M = k⊕r = Ext ⁡ d(k,M), where r is the rank of M. The latter is equal to k⊕r since R is Gorenstein.

Now, suppose that for any finite module N with projdim ⁡ N ≤ m − 1, Tor ⁡ i(k,N) = Ext ⁡ d−i(k,N). We can obtain such an N by taking a minimal free resolution of M

0 → Fm →⋯ → F → M → 0

then cutting it into

0 → Fm →⋯ → F1 → N → 0

and

0 → N → F → M → 0.

Note that since this resolution was minimal, N ⊂𝔪F – we’ll use this later. The first sequence implies that projdim ⁡ N ≤ m − 1, so we apply Hom ⁡ (k,−) and k ⊗− to obtain:

The green arrows are isomorphisms by our inductive hypothesis, and the equations and arrows in red hold since R is Gorenstein. Vertically adjacent modules between zeros are isomorphic by exactness, so the dashed isomorphisms are induced by the green arrows. This proves the desired equality for i ≥ 2, but we must be careful with the remaining two cases. The maps f and g exist since every other horizontal arrow in the diagram is an isomorphism, necessitating that each square is commutative. Having established that f and g exist, we can determine that they are isomorphisms by the five lemma, which proves the claim. □
